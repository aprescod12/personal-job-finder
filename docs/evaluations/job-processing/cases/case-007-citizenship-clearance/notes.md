# Case 007 Notes

This case provides an unambiguous hard-eligibility contrast to the ambiguous sponsorship case.

United States citizenship, ability to obtain and maintain a Secret clearance, and no visa sponsorship are explicit restrictions. They should be captured as blockers by the Processing Agent because the listing itself makes them mandatory.

An existing active clearance remains preferred. Conflating “able to obtain” with “already holds” would incorrectly reject otherwise eligible entry-level candidates.

This case also tests travel and driver's-license extraction.
