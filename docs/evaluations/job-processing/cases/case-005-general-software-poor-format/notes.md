# Case 005 Notes

This case intentionally removes conventional headings, punctuation, and sentence structure.

It tests whether extraction depends too heavily on clean employer templates. The expected title is normalized to title case, but all facts must still come from the compressed source.

The posting explicitly says a degree is not required and is silent on authorization. U.S. time-zone overlap must not be converted into an immigration restriction.

The contract is part-time for six months even though the primary employment enum can represent only `contract`; schedule and term remain supplemental.
