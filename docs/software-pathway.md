# Software Pathway

Software is an accepted technical direction within the industry-first career strategy.

Medical-device software, embedded systems, firmware, controls, software verification and validation, test automation, software quality, integration, reliability, cybersecurity, and product-security roles can be priority or adjacent opportunities when they support entry into medical devices or healthcare technology.

General software roles outside the target industry are still evaluated on skills, but they do not automatically outrank strong medical-device opportunities.
