# Software Pathway

Software is an accepted technical direction within the industry-first career strategy.

## Recognized directions

- Medical-device software engineering
- Embedded software and embedded systems
- Firmware engineering
- Controls and robotics software
- Software verification and validation
- Software test automation
- Software quality and design assurance
- Software integration and reliability
- Cybersecurity and product security for connected devices

A direct medical-device software match can be classified as a priority role. A related software function inside medical devices or healthcare technology can be classified as an adjacent opportunity.

General software roles outside the target industry still receive normal skill, education, and experience scoring, but they do not automatically outrank strong MedTech opportunities.
